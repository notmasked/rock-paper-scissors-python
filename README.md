# Rock, Paper, Scissors

  A terminal game of Rock Paper and Scissors which has persistent wins and losses, a computer opponent and easy to understand controls.

## Features

* Play against the computer
* Random computer moves
* Persistent win/loss tracking
* View lifetime game statistics
* Main menu navigation
* Basic input validation

## How to Run

1. Make sure Python 3 is installed.
2. Clone this repository or download the source code.
3. Ensure `wins.txt` and `losses.txt` are present in the project directory.
4. Open a terminal in the project folder.
5. Run:

```bash
python main.py
```

## Controls

### Main Menu

* **1** - Play
* **2** - View Statistics
* **3** - Exit

### During Gameplay

* **1** - Rock
* **2** - Paper
* **3** - Scissors
* **4** - Back to Main Menu

## Files

* `main.py` – Game logic
* `wins.txt` – Stores the total number of wins
* `losses.txt` – Stores the total number of losses
